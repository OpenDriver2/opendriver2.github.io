REDRIVER2 - open-source reimplementation of Driver 2
Game source code has been restored using original symbols
Significant improvements over original PS1 version including bug fixes and restoration of cut features
Improved visuals such as proper widescreen mode, increased draw distance and PGXP with Z-buffer
Native cross-platform - runs on Windows and Linux, planned in Web-browser play and backport to PS1
